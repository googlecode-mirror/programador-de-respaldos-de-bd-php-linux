# Introducción #

Este es un programa que se desarrolla con la finalidad de ayudar a programar tareas de respaldo desde una aplicación web, en un sistema con Linux y para que los respaldos sean guardados en una dirección ftp o dirección de windows dada.


# Instalación #

En el archivo fnc/Funciones.php tiene un constructor donde se debe cambiar la ruta de la aplicación.

Debe descomprimirse en una carpeta web.

# Requisitos #

- crontab
- ftp
- smbclient
- webserver
- php
- mysql
